# ccube FDS Web App Builder

Comprehensive guidance for building web applications that strictly follow the Flagship Design System (FDS).

## What it does

Enforces FDS components, tokens, and accessibility patterns across every Copilot suggestion, ensuring consistent, compliant output for web application scaffolding and development.

## Skills

### `cc-vite-react-ds`

Activated when a user wants to create or build a new web application using the Flagship Design System. Provides guidance on:

- Tech stack — Vite, React, TypeScript
- Design system — FDS components, design tokens, spacing, colour, and typography
- Accessibility — WCAG compliance patterns enforced by default
- Component patterns — FDS-compliant compositions and usage patterns
